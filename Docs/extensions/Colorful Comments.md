# Blocks
---

#### Reporters
Includes a string input and a comment input. The block returns the string input.

`Reporter`

#### Booleans
Includes a boolean input and a comment input. The block returns the boolean input.

`Boolean`

#### Blocks that do nothing:
These blocks do nothing dog...
