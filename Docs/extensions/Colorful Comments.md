# Colorful Comments
---
### Description
A colorful way to organize your projects!
#### Plans:
- A way to show and hide individual colors.
- Long Term: A way for user's to add their own colors.

# Blocks
---

#### Reporters
Includes a string input and a comment input. The block returns the string input.

`Reporter`

#### Booleans
Includes a Boolean input and a comment input. The block returns the Boolean input.

`Boolean`

#### Blocks that do nothing:
These blocks do nothing dog...
- **hat**
- **command**
- **end**
